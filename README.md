# omgdb
A plugin and extension manager for GDB.

# Goals 
Provide a simple, easy to use framework for extending, configuring, and enhancing gdb.

# Plugins
- [ ] Powerline Prompt
- [ ] Stack Frame prompt
- [ ] Simple Prompt
- [ ] TUI Layout
- [ ] Inputrc bindings
- [ ] C++ pretty printer
